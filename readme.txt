Adds Google Plus One Button to posts.

== Description ==
This plugin adds Google Plus One Button to every post on your WordPress Site.
This plugin is simple and uses in built function to display Google Plus 
One Button below every post. The size property of the plugin is set to "standard"
and the annotation property is set to "inline". Read about these properties here
https://developers.google.com/+/plugins/+1button/

== Installation ==
1. Download the plugin
1. Unzip the plugin and upload to /wp-contents/plugins/
1. Navigate to "Plugins" panels.
1. Activate the plugin.
